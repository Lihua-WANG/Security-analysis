Program structure:
data: Data preprocessed from raw data for both tasks.
src: The source code files to implement task I and task II.
notebook: The results represented by jupyter notebook.
outputs: It has images and cv fils with all outputs from models for each of the flows and clustering techniques used.

Run the program:
1. For task I, please run the main.py
2. For task II, please run the main2.py
